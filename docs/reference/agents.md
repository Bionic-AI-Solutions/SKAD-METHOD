## Default Agents

This page lists the default BMM (Agile suite) agents that install with SKad Method, along with their skill IDs, menu triggers, and primary workflows. Each agent is invoked as a skill.

## Notes

- Each agent is available as a skill, generated by the installer. The skill ID (e.g., `skad-dev`) is used to invoke the agent.
- Triggers are the short menu codes (e.g., `CP`) and fuzzy matches shown in each agent menu.
- QA (Quinn) is the lightweight test automation agent in BMM. The full Test Architect (TEA) lives in its own module.

| Product Manager (John)      | `skad-pm`            | `CP`, `VP`, `EP`, `CE`, `IR`, `CC` | Create/Validate/Edit PRD, Create Epics and Stories, Implementation Readiness, Correct Course        |
| Architect (Winston)         | `skad-architect`     | `CA`, `IR`                         | Create Architecture, Implementation Readiness                                                       |
| Scrum Master (Bob)          | `skad-sm`            | `SP`, `CS`, `ER`, `CC`             | Sprint Planning, Create Story, Epic Retrospective, Correct Course                                   |
| Developer (Amelia)          | `skad-dev`           | `DS`, `CR`                         | Dev Story, Code Review                                                                              |
| QA Engineer (Quinn)         | `skad-qa`            | `QA`                               | Automate (generate tests for existing features)                                                     |
